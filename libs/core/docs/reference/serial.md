# Serial

Reading and writing data over a serial connection.

```cards
serial.writeLine("");
serial.writeNumber(0);
serial.writeValue(x, 0);
serial.writeString("");
```

### Advanced

```cards
serial.writeBuffer(pins.createBuffer(0));
```
